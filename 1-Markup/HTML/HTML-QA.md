# HTML – Interview Q&A

## Basics

**Q:** What is HTML?
**A:** HyperText Markup Language used to structure content on the web.

**Q:** What is a tag?
**A:** A markup element used to define HTML structure.

**Q:** What is an element?
**A:** A complete HTML unit consisting of opening tag, content, and closing tag.

**Q:** What is an attribute?
**A:** Extra information added inside a tag to modify element behavior.

---

## Structure

**Q:** What is `<!DOCTYPE html>`?
**A:** Declares the document type and tells the browser to use HTML5.

**Q:** Purpose of `<html>` tag?
**A:** Root element that wraps the entire HTML document.

**Q:** Difference between `<head>` and `<body>`?
**A:** `<head>` contains metadata, `<body>` contains visible page content.

---

## Semantic HTML

**Q:** What is semantic HTML?
**A:** HTML elements that clearly describe their meaning.

**Q:** Examples of semantic tags?
**A:** `header`, `nav`, `main`, `section`, `article`, `footer`.

**Q:** Why use semantic HTML?
**A:** Improves accessibility, SEO, and code readability.

---

## Forms

**Q:** Purpose of `<form>` element?
**A:** Collects user input and sends data to a server.

**Q:** Difference between `GET` and `POST`?
**A:** GET sends data in URL, POST sends data in request body.

**Q:** What is the `<label>` element?
**A:** Associates text with a form input for accessibility.

---

## Media

**Q:** Purpose of `<img>` tag?
**A:** Embeds images in a webpage.

**Q:** What is `alt` attribute?
**A:** Alternative text describing an image for accessibility.

**Q:** Difference between `<audio>` and `<video>`?
**A:** `<audio>` plays sound, `<video>` plays video media.

---

## Miscellaneous

**Q:** What is the difference between `div` and `span`?
**A:** `div` is block-level, `span` is inline.

**Q:** What are block elements?
**A:** Elements that take full width and start on a new line.

**Q:** What are inline elements?
**A:** Elements that flow within text without starting a new line.

---

## Quick Recall

- HTML → Structure of web pages
- Semantic tags → Meaningful structure
- Forms → User input collection
- Accessibility → Important for usability and SEO
